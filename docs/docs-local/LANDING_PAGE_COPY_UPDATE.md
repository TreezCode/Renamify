## Conversion Optimization Notes

### Hierarchy of value propositions (in order of conversion weight)
1. **AI** — headline hook, "I've never seen a renamer do this"
2. **SEO** — concrete business outcome, "my images will rank better"
3. **Platform-specific** — validates it's built for them (not generic)
4. **Time saved** — emotional benefit, "I won't spend 3 hours on this"
5. **Professional CSV workflow** — signals enterprise-grade tool

### Moments of maximum upgrade intent to reinforce
- When free user hits the 20-image limit → **AI features gate message + upgrade CTA**
- When user sees SEO score is red → **"AI can fix this — upgrade to Pro"**
- After first successful export → **"Add AI alt text next time — upgrade to Pro"**
- On pricing page below free tier → **"Your competitors are using AI alt text"**

### Copy to avoid
- ❌ "Simple batch renamer" — undersells, attracts wrong user
- ❌ "AI-powered renaming" — too generic, doesn't explain the SEO angle
- ❌ Any mention of "free tool" without the Pro upgrade path visible
- ❌ "RAW support" as a headline feature — table stakes for photographers, not e-commerce primary
